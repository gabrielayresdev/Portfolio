# Portfolio

## Sobre

Portfólio desenvolvido com React, TypeScript e Sass.

Caso queira utilizar o site, acesse https://gabriel-ayres.vercel.app

![Site responsivo. Design adaptável para todas as plataformas](/src/assets/readme_images/template.png)

## Tecnologias

Essas foram as tecnologias usadas no projeto:

- React
- Typescript
- Sass

## Funcionalidades

As principais funcionalidades são:

- Scroll suave
- Envio de emails pelo formuário de contato
- Validação de formulário
